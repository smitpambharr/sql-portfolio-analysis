## 📌 Project Overview

A SQL-driven investment portfolio analysis built for a fictional UHNW (ultra-high-net-worth) client at "Golden Gate Capital." The project covers a five-asset ETF portfolio — designing the schema, writing analytical queries, and producing a full investment analysis report with performance metrics, risk assessment, and optimization recommendations.

---

## 🎯 Business Problem

Portfolio managers need to understand performance across assets, time windows, and risk profiles. This project asks: *how has this five-asset portfolio performed, what does the risk-return profile look like, and where are the optimization opportunities?*

---

## 🔍 Analysis Approach

- Relational database schema designed and implemented in MySQL Workbench
- Stored procedures written with parameterized time windows for flexible analysis
- Portfolio performance metrics calculated across assets and time periods
- Risk-return analysis using SQL aggregate and window functions
- Findings compiled into a structured analysis document with recommendations
